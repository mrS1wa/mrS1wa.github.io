Eagle Lake is a calligraphy font-style created by Astigmatic. It is a newer font that takes heavy inspiration from the Running Book Hand style of calligraphy. Like many calligraphy fonts, the bars and arms tend to be quite thin while the spines and tails of the letterform are thick. A key feature of this font is the smaller capital lettering which allows for a "taller looking x-width". This taller x-width allows for better readability and clarity.
